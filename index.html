Installer sur Apple Watch
Apple ne permet pas les cadrans tiers natifs. Trois méthodes disponibles selon vos besoins.
📸
Cadran Photos
Rapide · Natif iOS
  • 1Sélectionnez votre modèle, tapez 📲 Montre — PNG à résolution exacte téléchargé.
  • 2Envoyez via AirDrop sur iPhone → sauvegardez dans Photos.
  • 3App Montre → Cadrans → + → Cadran Photos.
  • 4Sélectionnez l'image → Ajoutez → synchro auto.
⚠ Affiche le fond avec les aiguilles natives Apple Watch par-dessus.
⌚
Clockology
Cadran complet · App tierce
  • 1Installez Clockology (App Store) sur iPhone et montre.
  • 2Exportez le PNG et le SVG depuis les boutons d'export.
  • 3Créez un cadran → importez PNG comme fond.
  • 4Ajoutez les aiguilles depuis la bibliothèque ou importez le SVG (Pro).
  • 5Send to Watch pour synchroniser.
💡 Clockology Pro : import SVG des aiguilles pour un résultat pixel-perfect.
🌐
Export HTML animé
Standalone · Partout
  • 1Tapez 🌐 HTML — un fichier HTML autonome est généré avec l'horloge animée complète.
  • 2Ouvrez-le dans n'importe quel navigateur (local ou hébergé).
  • 3Intégrable dans une WebApp, un TWA Android ou Clockology via WebView.
